美团商业分析大赛 - 数据分析案例说明

一、项目背景
本案例围绕小象超市即时零售门店规划问题，基于门店分站龄订单规模表与门店基础信息表，完成门店销售规模关键因子识别、成长机制拆解和候选点位选址分析。

二、核心分析内容
1. 数据清洗与整理：将原始双层表头数据拍平，按站点ID构建门店静态因子表、结果表与合并底表。
2. 特征工程：构造高等级月活人数、距离加权覆盖户数、即时竞争强度、近/中/远场户数占比等变量。
3. 探索性分析：分析M12日均单量分布、长尾特征及各候选因子与M12的单变量关系。
4. 成长机制分析：按站点拟合“日均单量 = b + k × 站龄月”，将M12拆解为初始流量b与增长率k。
5. 模型应用：使用二次项模型进行解释，使用树模型辅助预测候选站点销售规模与优先级排序。

四、简历可概括
