# 美团商业分析大赛：即时零售门店销售规模分析与选址优化

## 项目简介

本项目为美团商业分析大赛数据分析案例，研究主题为小象超市即时零售门店销售规模影响因素识别与选址优化。项目基于门店经营数据、用户画像数据和竞对环境数据，完成数据清洗、特征构建、关键变量识别、成长机制分析及模型结果输出。

## 项目目标

1. 识别影响门店销售规模的关键因子；
2. 构建门店销售规模分析框架；
3. 分析门店从开业到成熟期的增长机制；
4. 基于模型结果辅助候选站点排序与选址判断。

## 分析方法

- 数据清洗与建模底表构建
- 单变量分析与 Spearman 相关分析
- 分箱均值分析
- 二次项回归模型
- 树模型
- 站点成长率线性回归分析

## 核心发现

1. 门店销售规模并非单纯由覆盖人口决定，而更依赖高质量用户规模与有效近场覆盖；
2. 高等级月活人数、距离加权覆盖户数、0~1.5km 覆盖户数是关键驱动因子；
3. 门店成熟期规模可以拆解为初始流量 b 与增长率 k；
4. 距离加权覆盖户数主要影响起步流量，高等级月活人数主要影响成长速度；
5. 二次项模型适合解释变量作用机制，树模型适合辅助预测与站点排序。
